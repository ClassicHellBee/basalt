Welcome to the Basalt Linux project, a source-based distro.

Basalt is a small stage 3 build of the lfs project up to chapter 8 of the installation. It also includes a small package manager of the same name.

2025-07-25: Finally, we're happy to announce that we've updated the handbook and released stage 3 with the package manager. The documentation is still being created... :)

2025-07-28: From the Basalt project, we would like to announce that we have now released the image with our Basalt package manager, correcting errors from previous releases that prevented it from running correctly.
